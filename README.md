
# Alarm Clock

Make a simple Alarm Clock with HTML, CSS and JavaScript which takes <br>
• Hours (01-12) <br>
• Minutes (00-59) <br>
• Seconds (00-59) <br>

and then set the alarm showing the time below near the clock
